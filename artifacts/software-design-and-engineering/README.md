# Software Design and Engineering Artifact

This folder contains my Software Design and Engineering artifact from CS 405.

## Overview
This artifact is an encryption and decryption program written in C++. It reads a text file, encrypts it using a repeating-key XOR algorithm, and then decrypts it back to readable text.

## Enhancements
- Improved error handling
- Removed hardcoded encryption key
- Improved file validation
- Increased modularity and readability
