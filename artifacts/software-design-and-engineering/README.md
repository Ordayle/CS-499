# Software Design and Engineering Artifact

This folder contains my Software Design and Engineering artifact from CS 405.

## Overview
This artifact is a C++ encryption and decryption program. It reads a text file, encrypts the contents using a repeating-key XOR algorithm, and then decrypts the data back into readable text.

## Enhancements
- Improved error handling for file input and user input
- Removed the hardcoded encryption key and replaced it with user input
- Added stronger file validation to prevent runtime issues
- Refactored the code to improve modularity and readability
